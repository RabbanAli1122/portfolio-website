# Rabban Ali — Portfolio

Personal portfolio website built with plain HTML, CSS, and JavaScript. No frameworks, no build tools.

## Pages

- **Work** — project index with links to GitHub repos
- **About** — skills, education, and experience
- **Goals** — upcoming projects and future plans
- **Contact** — links and a mailto-based contact form

## Features

- Light / dark theme toggle (saved to localStorage)
- Responsive layout for mobile and desktop
- Single-file, zero dependencies

## Stack

HTML · CSS · Vanilla JavaScript · Google Fonts (Cormorant Garamond, DM Mono)

## Usage

Just open `index.html` in a browser. No setup needed.
